<dt><a name="entity">   </a>Entity      <dd>An <a href="#fps">FPS-game</a> entity, is an object which the player can interact with. Most entities are monsters, weapons and moving doors. Some entities are used to light the level as if you placed an invisible light-bulb in a spot. Other entities, like doors, require that it has some <a href="#polyhedron">polyhedrons</a> connected to it.
