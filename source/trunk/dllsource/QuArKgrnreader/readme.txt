target grnreader.exe

Release grnreader.exe for use with QuArK 6.6.0 Beta 4
can be built with these files and Visual Studio 2005.

Works with granny2.dll included and possibly with later versions.

Place grnreader.exe and granny2.dll in QuArk's dlls folder
and use QuArK's Model Editor, use "new model" icon button to start editor,
delete empty "Component",
then click main menu's File > Model Importers > .gr2 Importer
and select model. (only works for static models and bones, no animation)
